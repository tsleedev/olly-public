# 감사 일기 (Gratitude Journal) 상세 분석

## 1. 경쟁 앱 분석

### 1.1 Gratitude: Self-Care Journal
- **링크**: https://apps.apple.com/us/app/gratitude-self-care-journal/id1372575227
- **가격**: 무료 + 인앱구매
  - Monthly: $4.99
  - Annual: $29.99
  - Lifetime: $39.99
- **평점**: ⭐ 4.9 (45K 리뷰) - 최다 리뷰 & 최고 평점
- **주요 기능**:
  - 감사 일기 (사진 & 음성)
  - 긍정 확언 (Affirmations)
  - 비전보드 생성
  - Daily Zen (명언, 동기부여)
  - 21일 챌린지
  - 저널 프롬프트 제공
- **사용자 불만**:
  - 영어 전용
  - 일부 콘텐츠 유료 제한
  - 알림 설정 복잡

### 1.2 365 Gratitude Journal
- **링크**: https://apps.apple.com/us/app/365-gratitude-journal-selfcare/id1072397377
- **가격**: 무료 + 인앱구매

  - 커뮤니티 모더레이션 부족
  - 앱 속도 느림

### 1.3 Grateful: A Gratitude Journal
- **링크**: https://apps.apple.com/us/app/grateful-a-gratitude-journal/id1197512462
- **가격**: 무료 + 인앱구매
  - One-time: $4.99
  - Monthly: $3.99
  - Annual: $14.99-29.99
- **평점**: ⭐ 4.6 (3K 리뷰)
- **주요 기능**:
  - 매일 다른 프롬프트

  - 디자인 다소 구식
  - 위젯 없음

### 1.4 Happyfeed: Gratitude Journal
- **링크**: https://apps.apple.com/us/app/happyfeed-gratitude-journal/id707073943
- **가격**: 무료 + 인앱구매
  - Monthly: $3.99
  - Annual: $39.99
- **평점**: ⭐ 4.8 (3.8K 리뷰)

  - 사진/비디오 일기

  - 비디오 10초 제한

### 1.5 Delightful: Gratitude Journal

## 2. 시장 분석

### 2.1 시장 규모 & 트렌드
| 지표 | 수치 |
|------|------|
| Mental Wellness App 시장 | $7.48B (2024) |
| 감사 저널 카테고리 | 전체의 약 8-12% |
| 예상 성장률 | 15%+ CAGR |
| 주요 시장 | 북미 > 유럽 > 아시아 |

### 2.2 주요 트렌드
- **"3 Good Things" 방식**: 하루 3가지 감사 기록이 표준
- **사진 중심**: 텍스트 + 이미지 결합
- **Throwback**: 과거 기록 회상 기능 인기
- **소셜 공유**: 가족/친구와 공유 기능
- **마인드풀니스 통합**: 명상, 확언과 결합

### 2.3 타겟 사용자
1. **주요 타겟**: 25-45세 여성
   - 자기계발에 관심
   - 정신 건강 관리 의식 높음
2. **부 타겟**: 부모/가족
   - 자녀 교육 목적
   - 가족 공유 원함
3. **니치 타겟**: 치료 중인 환자
   - 심리상담 보조 도구
   - CBT 기반 접근

## 4. 차별화 포인트

### 4.1 경쟁 앱 약점
| 앱 | 약점 |
|----|------|
| Gratitude | 영어 전용, 기능 과다 |
| 365 Gratitude | 가격 복잡, 느림 |
| Grateful | 디자인 구식, 위젯 없음 |
| Happyfeed | 소셜 강요, 기능 제한 |

### 4.2 차별화 전략
1. **🎯 원탭 기록**
   - 위젯에서 바로 감사 기록
   - 타이핑 최소화 (템플릿/음성)
   - 30초 안에 완료

2. **📱 위젯 중심 UX**
   - 오늘의 감사 확인
   - 퀵 입력 버튼
   - 스트릭 표시

3. **🇰🇷 한국어 최적화**
   - 한국어 프롬프트
   - 한국 문화 맞춤 예시
   - 한국어 명언/확언

4. **🌸 미니멀 디자인**
   - 기능 과다 X
   - 딱 필요한 것만
   - iOS 17 네이티브 느낌

5. **🔄 똑똑한 Throwback**
   - 1주/1달/1년 전 감사
   - 패턴 분석 (자주 감사하는 것)
   - 감사 캘린더 히트맵

## 7. 프롬프트 예시 (한국어)

### 7.1 일반 프롬프트
- 오늘 당신을 미소 짓게 한 것은?
- 오늘 고마웠던 사람은?
- 이번 주 가장 좋았던 순간은?
- 당연하게 여겼지만 감사한 것은?

### 7.2 상황별 프롬프트
- **월요일**: 새로운 한 주를 시작하며 기대되는 것은?
- **금요일**: 이번 주 나를 성장시킨 경험은?
- **비오는 날**: 비오는 날만의 좋은 점은?
- **힘든 날**: 어려움 속에서도 감사할 수 있는 것은?
