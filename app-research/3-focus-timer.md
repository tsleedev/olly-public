# 포모도로/집중 타이머 (Focus Timer) 상세 분석

## 1. 경쟁 앱 분석

### 1.1 Focus Keeper - Pomodoro Timer
- **링크**: https://apps.apple.com/us/app/focus-keeper-pomodoro-timer/id867374917
- **가격**: 무료 + 인앱구매
  - Monthly: $3.99-12.99
  - Annual: $34.99-69.99
  - Lifetime: $99.99-199.99
- **평점**: ⭐ 4.8 (31K 리뷰)
- **주요 기능**:
  - 커스터마이징 가능한 타이머
  - ADHD 친화적 설계
  - 앱 블로킹
  - 상세 통계
  - Apple Watch 지원
  - 홈/잠금 화면 위젯
- **사용자 불만**:
  - 앱 크기 큼 (633MB)
  - 가격 다양해 혼란
  - 프로 기능 너무 비쌈

### 1.2 Flow: Focus & Pomodoro Timer
- **링크**: https://apps.apple.com/us/app/flow-focus-pomodoro-timer/id1423210932
- **가격**: 무료 + 인앱구매
  - Monthly: $2.99
  - Annual: $17.99
  - Lifetime: $39.99
- **평점**: ⭐ 4.8 (1.7K 리뷰)
- **주요 기능**:
  - 미니멀 디자인
  - Live Activities & Dynamic Island
  - 앱/웹 블로킹
  - 메트로놈 (집중 사운드)
  - Apple Health/Calendar 연동
  - iCloud 동기화
- **사용자 불만**:
  - macOS 버전 별도 구매
  - 일부 버그 (중복 Live Activity)
  - 통계 기능 기본적

### 1.3 FocusPomo · Pomodoro Timer
- **링크**: https://apps.apple.com/us/app/focuspomo-pomodoro-timer/id1528322796
- **가격**: 무료 + 인앱구매 (평생 구매 중심)
- **평점**: ⭐ 4.7+
- **주요 기능**:
  - Interactive 위젯
  - Dynamic Island
  - StandBy 모드 지원
  - 앱 화이트리스트
  - Apple Health 연동
  - 수동 시간 추가
- **사용자 불만**:
  - UI 다소 복잡
  - 한글 지원 불완전

### 1.4 Pomodoro - Focus Timer (by delightroom)
- **링크**: https://apps.apple.com/us/app/pomodoro-focus-timer/id1265128036
- **가격**: 무료 + 인앱구매
- **주요 기능**:
  - iOS 14 위젯
  - Apple Watch Complications
  - 백색소음 플레이어
  - 커스텀 배경 이미지
  - 음성 피드백
- **사용자 불만**:
  - 업데이트 빈도 낮음
  - 최신 iOS 기능 미지원

### 1.5 Focus To-Do (Hybrid)
- **링크**: 여러 플랫폼 지원
- **가격**: 무료 + 구독
- **주요 기능**:
  - 포모도로 + 할일 관리 통합
  - 태스크별 시간 추적
  - 리마인더 & 데드라인
- **사용자 불만**:
  - 너무 복잡함
  - 단순 타이머 원하는 사용자에게 과함

### 2.2 주요 트렌드
- **Dynamic Island 활용**: 타이머 상태 상시 표시
- **앱 블로킹 통합**: 집중 모드와 연계

- **학습자 타겟**: 수험생, 대학생 주요 고객
- **원격근무 확대**: 재택 근무자 생산성 관리

### 2.3 타겟 사용자
1. **주요 타겟**: 학생 (고등학생, 대학생)
   - 시험 준비
   - 공부 시간 관리
2. **부 타겟**: 원격 근무자
   - 재택 근무 집중력
   - 업무 시간 추적
3. **니치 타겟**: ADHD 사용자
   - 구조화된 시간 관리
   - 짧은 집중 세션

## 4. 차별화 포인트

### 4.1 경쟁 앱 약점
| 앱 | 약점 |
|----|------|
| Focus Keeper | 비쌈, 앱 크기 큼 |
| Flow | macOS 별도, 통계 기본 |

| Pomodoro | 업데이트 느림 |

### 4.2 차별화 전략
1. **📱 위젯 퍼스트**
   - 앱 열 필요 없이 위젯에서 시작/정지
   - Interactive 위젯 (iOS 17+)
   - 진행 상황 실시간 표시

2. **🎯 극단적 단순함**
   - 설정 최소화
   - 버튼 하나로 시작
   - 불필요한 기능 배제

3. **🇰🇷 한국어 네이티브**
   - 완벽한 한국어 UI
   - 한국 학생 문화 반영 (수능 모드 등)

4. **💰 합리적 가격**
   - 학생 타겟 → 저렴하게
   - 평생 구매 $29.99

5. **🏝️ Dynamic Island 최적화**
   - 타이머 상태 상시 표시
   - 탭으로 빠른 컨트롤

6. **📊 게이미피케이션 라이트**
   - 일간/주간 목표
   - 연속 집중 스트릭
   - 간단한 성취 뱃지

## 6. 위젯 활용 아이디어

### 6.1 Small 위젯 (2x2) - Interactive
```
┌─────────────────┐
│    🍅 집중      │
│                 │
│    25:00       │
│                 │
│   [ ▶️ 시작 ]   │
└─────────────────┘
```

진행 중:
```
┌─────────────────┐
│    🍅 집중 중   │
│                 │
│    18:42       │
│   ━━━━━━━░░░    │
│   [ ⏸️ 일시정지 ]│
└─────────────────┘
```

### 6.2 Medium 위젯 (4x2)
```
┌───────────────────────────────────┐
│  🍅 집중 타이머                   │
│                                   │
│      25:00         [ ▶️ 시작 ]   │
│  ━━━━━━━━━━━━━━━━━               │
│                                   │
│  오늘: 🍅🍅🍅○○○ (3/6 완료)      │
└───────────────────────────────────┘
```

### 6.3 Large 위젯 (4x4)
```
┌───────────────────────────────────┐
│  🍅 오늘의 집중                   │
│                                   │
│         [ 25:00 ]                │
│         [ ▶️ 시작 ]               │
│                                   │
│  ━━━━━━━━━━━━━━━━━━━━━           │
```
